利用Python进行数据分析¶

开始之前¶

先介绍一下接下来会用到的几类库。

  • Numpy（Numerical Python）
    • 提供了高效的多维数组对象ndarray，效率至少是Python原生的5倍
    • 科学计算
    • 粘合剂
    • 数值型数据的处理效率比内置结构更高效
  • Pandas 熊猫包
  • Matplotlib
  • SciPy
    • 科学计算 虽然MATLAB很厉害，但是NP+SP可以完爆MATLAB

Anaconda + PyCharm

随机漫步的表示¶

随机森林是现代概率论中的重要组成部分，因此我们利用NumPy的随机数模块进行随机漫步的一个表示，事实上，我们在最初已经做过一个简单的例子。

In [48]:
import random
position = 0
walk = [position]
steps = 1000
for i in xrange(steps):
    step = 1 if random.randint(0,1) else -1
    position += step
    walk.append(position)
